🐍 HH_Django

Платформа поиска работы и анализа вакансий

🎓 Дипломный проект | Python-разработчик

📖 О проекте

🎯 Цель

Разработать веб-приложение для публикации вакансий и резюме с полноценным REST API и контейнеризацией для лёгкого развёртывания.

👥 Пользователи

  • Соискатели (поиск работы, отклики)
  • Работодатели (публикация вакансий)
  • Администраторы (модерация)

🛠 Технологический стек

Python 3.11+ Django 5.1 DRF 3.15 PostgreSQL SQLite (dev) Docker Docker Compose Nginx Gunicorn Pillow django-cleanup django-debug-toolbar Pytest / Coverage

📁 Структура приложений

  • hhapp — вакансии, резюме, API
  • userapp — кастомная модель пользователя
  • templates — HTML-шаблоны
  • static/media — статика и загрузки

⚙️ Основной функционал

🌐 Веб-интерфейс

  • Регистрация и авторизация (кастомная модель)
  • Публикация и редактирование вакансий
  • Создание и загрузка резюме (фото/файлы)
  • Админ-панель для управления контентом
  • Отправка email-уведомлений (консольный backend)

🔌 REST API

  • Пагинация (20 элементов на страницу)
  • Аутентификация: Basic + Session
  • Эндпоинты: /api/vacancies/, /api/resumes/
  • Фильтрация через django-filter
  • Поддержка Markdown для описаний

🐳 Docker: запуск проекта

🛠 Режим разработки

$ docker-compose up -d --build

$ docker-compose exec web python manage.py migrate

$ docker-compose exec web python manage.py createsuperuser

🌐 http://localhost:8000

🚀 Режим продакшена

$ docker-compose -f docker-compose.prod.yml up -d --build

$ docker-compose -f docker-compose.prod.yml exec web python manage.py migrate

$ docker-compose -f docker-compose.prod.yml exec web python manage.py collectstatic --noinput

🌐 http://localhost:1337

📋 Остановка и очистка

$ docker-compose down # Остановка dev

$ docker-compose -f docker-compose.prod.yml down # Остановка prod

$ docker-compose down -v # Полная очистка с удалением БД
